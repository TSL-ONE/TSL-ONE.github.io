Key Features

  • Word-level Thai Sign Language recognition dataset
  • Uncontrolled, naturalistic recording conditions
  • Standardized annotation process with automated validation
  • Comprehensive handedness analysis
  • Signer-independent evaluation protocol

Available Datasets

Skeletal Data

Processed skeletal keypoints extracted using MediaPipe for pose and hand landmark detection. With 54 landmarks (head landmarks including eyes, ears, and nose, 21 per hand) and a neck landmarks for SPOTER model.

Format: npy files

Availability: Available in this repository

Raw Video Files

Original video recordings in naturalistic settings with varying lighting and backgrounds. Complete video dataset for comprehensive research and alternative feature extraction methods.

Format: MP4 video files

Availability: Available upon request (Please fill in the contact form at the end of this page for non-commercial use)
